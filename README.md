# CodeAlpha - Banking System (C++)

## Description (TASK 3)
A simple banking system using OOP concepts in C++.

## Features
= Supports multiple customers and fund transfers between accounts.
- Customer and Account management
- Deposit and withdrawal
- Transaction history
- Balance display
  
## Concepts Used
- Classes and Objects
- Vectors
- Basic OOP principles

## How to Run
g++ bank_system.cpp -o bank
./bank
